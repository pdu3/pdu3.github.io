PathRecall: Memory-Augmented Visual Question Answering for Indoor Navigation
Designed and implemented a retrieval-augmented visual question answering (VQA) system to assist indoor navigation for visually impaired users. Integrated temporal memory using a custom MemoryNet to retrieve relevant frames from navigation history via BLIP-2 + MiniLM embeddings and a FAISS vector store. Enhanced VQA performance by combining retrieved frames with a question-aware captioning module, enabling context-aware answers across time. Achieved top-1 frame recall of XX% and outperformed BLIP-2 and MiniGPT-4 baselines on a custom indoor benchmark.
