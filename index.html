PathRecall: Memory-Augmented Visual Question Answering for Indoor Navigation

Designed and implemented a retrieval-augmented visual question answering (VQA) system to assist indoor navigation for visually impaired users. Integrated temporal memory using a custom MemoryNet to retrieve relevant frames from navigation history via BLIP-2 + MiniLM embeddings and a FAISS vector store. Enhanced VQA performance by combining retrieved frames with a question-aware captioning module, enabling context-aware answers across time. Achieved top-1 frame recall of 85.5% and outperformed BLIP-2 and MiniGPT-4 baselines on a custom indoor benchmark.

AR-based Automated Annotation Tool for Indoor Navigation Assistance

This project explores how Augmented Reality (AR) can enhance indoor navigation and object recognition for people with low vision. We developed a mobile application that automatically detects and labels salient objects (e.g., desks, chairs, low ceilings) using object detection, AR labeling, and precise indoor positioning. Unlike existing tools such as NearbyExplorer, which depend on large teams for manual annotation, our system enables a single volunteer to populate a shared database in real time. Low vision users can then receive instant audio descriptions of surrounding objects via their smartphones, reducing the effort required for orientation and improving independence in diverse indoor environments.

Legal RAG Assistant – Retrieval-Augmented Legal QA System (Course Project)

Built a domain-specific RAG pipeline for U.S. legal documents, fine-tuning Qwen2.5-Instruct with LoRA for improved legal text comprehension. Developed a ChromaDB-based retrieval layer integrated with multi-turn Q&A and compliance checks. Achieved 95% clause interpretation accuracy and <0.5s response time in prototype testing.
